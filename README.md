# 前端作品集 Demo

这是一个纯 HTML/CSS/JavaScript 构建的前端作品集页面。适合用于面试演示和个人项目展示。

## 功能亮点

- 响应式布局，支持 PC 和移动端
- 顶部导航与移动端菜单
- 深色模式切换并记住用户偏好
- 项目详情弹窗与项目卡片交互
- 页面滚动显现动画
- 可通过浏览器直接打开运行

## 运行方式

1. 打开 `index.html`
2. 在浏览器中预览
3. 点击顶部主题按钮切换深/浅色模式
4. 在“项目展示”区域点击“查看案例”查看项目详情弹窗

## 自定义内容

请根据你的个人信息替换以下内容：

- `index.html` 中的姓名、简介、联系方式
- `script.js` 中的 GitHub 链接
- 项目描述和标签为你真实的项目经验

## 部署到 GitHub Pages

1. 将 `portfolio-website` 文件夹提交到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 `main` 分支下的 `/root` 作为发布源
4. 保存后等待几分钟，GitHub 会生成你的演示网址
